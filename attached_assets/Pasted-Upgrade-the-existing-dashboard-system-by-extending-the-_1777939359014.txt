FEATURE 1: SMART SHIFT-END REMINDER (BASED ON EXISTING SETTINGS)

Use the existing "reminder minutes" setting (s-reminder input).

Implement a background checker:

* Run every 60 seconds using setInterval
* For each member:

  * Get shiftEnd
  * Calculate remaining time
  * If remaining time <= reminder minutes:
    → Find all tasks assigned to this member
    → Filter tasks where status = "inprogress"
    → If tasks exist:
    → Trigger webhook

Prevent duplicate notifications:

* Add flag: member.shiftReminderSent
* Do not send again if already sent
